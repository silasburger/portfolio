Window Restore

Script for macOS via Hammerspoon, Lua

I encountered an issue where macOS apps use different methods to open their primary windows. My workflow involves switching apps, opening a new window if needed, all via keyboard. However, I found that each app had its own keystroke for opening a primary window, or sometimes none at all. To avoid hunting for app icons in the dock, I created a Lua script with Hammerspoon that opens the primary window of any app with just three keystrokes.
